# repogithub_tom

Gemeinsames Arbeitsverzeichnis für Claude Code Sessions (lokal ↔ Remote).

## Workflow

```bash
cd ~/repogithub_tom
claude          # Claude Code lokal starten
git push        # Änderungen hochladen → tauchen in Remote-Session auf
```

## Inhalt

- Skripte, Pläne und Automatisierungen aus Claude-Sessions
- Inbox-Triage-Logs, Mail-Setup, Kalender-Konfiguration
